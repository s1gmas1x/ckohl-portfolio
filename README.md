# ckohl-portfolio

Personal portfolio and engineering showcase for Chad Kohl.

This project serves as the foundation for:

- Professional portfolio site
- Technical case studies
- Project showcase
- Experimental UI/animation work
- Future "works" client/services platform

## Goals

The purpose of this project is to:

- Present technical experience professionally
- Showcase real-world projects and troubleshooting skills
- Practice modern engineering workflows
- Improve GitHub, branching, PR, and issue management skills
- Experiment with AI-assisted development workflows

## Planned Sections

- Hero / Introduction
- Projects
- Technical Case Studies
- Skills & Experience
- Contact
- Future Works/Services Area

## Tech Stack

- Vue 3
- Quasar Framework
- Vite
- Vue Router
- GSAP (planned)
- Three.js (planned)

## Development Workflow

This project follows a lightweight agile-style workflow:

- GitHub Issues for tickets/tasks
- Feature branches for development
- Pull Requests for review
- Incremental MVP-focused development

## Project Status

Early development / foundation phase.

## Local Development

Install dependencies:

```bash
npm install
```

Start development server:

```bash
quasar dev
```

Build for production:

```bash
quasar build
```
